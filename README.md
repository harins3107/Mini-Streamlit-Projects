# Streamlit Practice Apps

A collection of small **Streamlit applications** built while learning Streamlit and experimenting with interactive web app development.

These mini projects demonstrate different Streamlit components, layouts, widgets, and Python concepts through simple, hands-on applications.

---

## 🚀 Overview

This directory contains beginner-to-intermediate Streamlit projects covering topics such as:

- Interactive widgets
- Session state
- Navigation between screens
- Data structures visualization
- Basic games
- Calculator applications

The goal of these projects is to explore Streamlit's capabilities while improving Python development skills.

---

## 📂 Projects

| File | Description |
|------|-------------|
| `calculator.py` | A basic calculator with arithmetic operations. |
| `double.py` | A simple interactive app demonstrating user input and calculations. |
| `multiscreen.py` | Demonstrates navigation across multiple Streamlit pages/screens. |
| `rocket.py` | A fun interactive rocket-themed mini application/game. |
| `screenbutton.py` | Demonstrates button interactions and page updates. |
| `stack.py` | Stack (LIFO) visualization with push and pop operations. |

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/harins3107/Mini-Streamlit-Projects.git
```

Navigate to the project

```bash
cd Mini-Streamlit-Projects/mock-ideas/streamlit
```

Install Streamlit

```bash
pip install streamlit
```

---

## ▶ Running an Application

Run any project using:

```bash
streamlit run calculator.py
```

or

```bash
streamlit run rocket.py
```

or any other file in this directory.
